# NadsFlip dApp - Monad Testnet Coin Flip Game

A decentralized coin flip game running on the Monad Testnet blockchain. This dApp allows users to bet on coin flips with a provably fair outcome.

## Features

- Connect with multiple wallets (MetaMask, Phantom, Rabby, Backpack)
- Place bets on Heads or Tails
- Animated coin flip visualization
- Transaction history tracking
- Responsive design for mobile and desktop
- Real-time balance updates

## Getting Started

### Prerequisites

- Node.js (v14.0.0 or later)
- npm or yarn
- A browser wallet extension (MetaMask, Phantom, Rabby, or Backpack)
- Some MON testnet tokens (available from the Monad faucet)

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/nadsflip-dapp.git
   cd nadsflip-dapp
   ```

2. Install dependencies
   ```bash
   npm install
   # or
   yarn install
   ```

3. Start the development server
   ```bash
   npm start
   # or
   yarn start
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Deployment

This project is configured for deployment on Vercel:

1. Push your code to a GitHub repository
2. Connect your repository to Vercel
3. Deploy with default settings
